------- English ----------

#For installing just type the following commands

	cd .../tictactoe
	make
	sudo make install
	make clean

#Done, now type

	jogodavelha

#To play and have fun!

#If you want to remove the game completely use

	sudo make remove

#Or manually remove jogodavelha from /usr/local/bin as well as the original files

------- Portugues --------

#Para instalar digite os seguintes comandos no terminal:

	cd .../tictactoe
	make
	sudo make install
	make clean

#Pronto, agora basta digitar

	jogodavelha

#para jogar.

#Para remover o jogo basta ir para o mesmo diretório pelo terminal e digitar

	sudo make remove

ou então remover o arquivo "jogodavelha" da pasta /usr/local/bin bem como a pasta
jogodavelha (se quiser remover completamente).
