------- English ----------

#For installing just type the following commands

	cd .../tictactoe
	make
	sudo make install
	make clean

#Done, now type

	tictactoe

#To play and have fun!

#If you want to remove the game completely use

	sudo make remove

#Or manually remove tictactoe from /usr/local/bin as well as the original files
if you want to remove it completely.

------- Portugues --------

#Para instalar digite os seguintes comandos no terminal:

	cd .../tictactoe
	make
	sudo make install
	make clean

#Pronto, agora basta digitar

	tictactoe

#para jogar.

#Para remover o jogo basta ir para o mesmo diretório pelo terminal e digitar

	sudo make remove

ou então remover o arquivo "tictactoe" da pasta /usr/local/bin bem como a pasta
tictactoe (se quiser remover completamente).
